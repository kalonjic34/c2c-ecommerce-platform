# C2C E-Commerce Platform

## Overview
A prototype Customer-to-Customer (C2C) e-commerce platform built with HTML, CSS, JavaScript, PHP, and MySQL.  
Includes:
- Main website for buyers and sellers
- Admin website with Role-Based Access Control (RBAC)
- Responsive design for desktop, tablet, and mobile

## Features
- User registration and login
- Product listing and detail pages
- Shopping cart and checkout
- Admin dashboard with CRUD for users and products
- MySQL database schema with roles (admin, seller, buyer)

## Tech Stack
- **Frontend**: HTML, CSS, Bootstrap, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Hosting**: Online server (InfinityFree/000webhost)
